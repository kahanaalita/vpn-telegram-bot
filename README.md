# VPN Telegram Bot

Telegram бот для получения VLESS конфигураций из GitHub репозитория.

## Команды

- `/start` - Приветствие
- `/configs` - Получить список VLESS конфигураций

## Запуск локально

```bash
export TELEGRAM_BOT_TOKEN="ваш_токен"
go run main.go
```
